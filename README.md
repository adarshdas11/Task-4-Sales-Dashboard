# Task 4 – Sales Dashboard (Data Analyst Internship)

## Objective
Create an interactive Sales Dashboard using Power BI to analyze business performance.

## Insights Delivered
• Total Sales, Orders, Quantity & AOV KPIs  
• Sales trend over time  
• Country-wise performance using map visualization  
• Sales distribution by product line  
• Deal-size performance (Small/Medium/Large)  
• Interactive filters (Country, Year, Product Line, Deal Size)

## Tools Used
• Microsoft Power BI  
• Power Query  
• Kaggle Dataset (Sales Data Sample)
